# Vessapaperien hintavertailu

UX/UI konseptiprojekti suomalaiselle vessapaperien hintavertailupalvelulle.

## Ongelma

Vessapaperipakkausten vertailu on vaikeaa, koska pakkauksissa on eri määrä rullia, arkkeja ja kerroksia. Pelkkä paketin hinta ei kerro todellista arvoa.

## Ratkaisu

Palvelu vertailee tuotteita hinnalla **hinta/100 arkkia**, mikä mahdollistaa tuotteiden todellisen vertailun.

## Keskeiset ominaisuudet

- tuotevertailu
- suodattimet
- tarjoussivu
- käyttäjäarvostelut
- badge-järjestelmä

Badget:
- Vastuullinen valinta
- Laatuluolalle
- Budjettipyllylle
